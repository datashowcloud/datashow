Preparo para Certificação AWS Certified Cloud Practitioner
Cada simulado abaixo é um conjunto completo de perguntas 100% aderentes ao exame de Cetificação AWS Certified Cloud Practitioner.
É necessário obter nota mínima de 70% de acerto para ser aprovado.
Atualmente o exame é realizado em Inglês.
Cada pergunta pode ter uma ou mais respostas.
01. Which AWS services can be used to store files?
a) Amazon CloudWatch
b) Amazon Simple Storage Service - S3
c) Amazon Elastic Block Store - EBS
d) AWS Config
e) Amazon Athena
02. Which of the following services uses AWS Edge Locations?
a) Amazon Virtual Private Cloud - VPC
b) Amazon CloudFront
c) Amazon Elastic Compute Cloud - EC2
d) AWS Storage Gateway
03. Which of the following is a benefit of Amazon Elastic Compute Cloud (EC2) over on-premise physical servers?
a) Automated backup
b) Paying only for what you used
c) The ability to choose hardware vendors
d) Root / Administrator Access
04. Which AWS service provides infrastructure security optimization recommendation?
a) AWS Application Programming Interface
b) Reserved Instances
c) AWS Trusted Advisor
d) Amazon Elastic Compute Cloud - Spot Fleet
05. Which service allows for the collection and tracking of performance metrics for AWS services?
a) Amazon CloudFront
b) Amazon CloudSearch
c) Amazon CloudWatch
d) Amazon Machnhine Learning
06. A company needs to know which user was responsible for terminating several critical Amazon Elastic Compute Cloud EC2 Instances. Where can the customer find this information?
a) AWS Trusted Advisor
b) Amazon EC2 instance usage report
c) Amazon CloudWatch
d) Amazon Cloud Trail logs
RESPOSTA 01
A resposta correta é: b) e c)
b) Amazon Simple Storage Service - S3 é um serviço de armazenamento de objetos. (explicação)
c) Amazon Elastic Block Store - EBS oferece volumes de armazenamento em bloco(disco rígido) para usar com instâncias do EC2, podendo criar sistema de arquivos. (explicação)
RESPOSTA 02
A resposta correta é: b) Amazon CloudFront
Edge locations(Pontos de presença), são infraestruturas da AWS enxutas, que rodam alguns poucos serviços, como: CloudFront, Route53 e AWS WAF. (explicação)
RESPOSTA 03
A resposta correta é: b) Paying only for what you used
RESPOSTA 04
A resposta correta é: c) AWS Trusted Advisor
O AWS Trusted Advisor é um aplicativo que inspeciona seu ambiente da AWS e faz recomendações para economizar, melhorar a performance do sistema ou corrigir falhas de segurança. (explicação)
